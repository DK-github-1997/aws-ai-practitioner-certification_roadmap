## 📖 Topics

### 1. What is Cloud Computing?
Cloud computing = using someone else's computers (servers) over the internet — paying only for what you use.

**3 Cloud Models:**
| Model | Full Form | Example | You Manage |
|-------|-----------|---------|-----------|
| IaaS | Infrastructure as a Service | EC2 | OS, App, Data |
| PaaS | Platform as a Service | Elastic Beanstalk | App, Data |
| SaaS | Software as a Service | Gmail, Salesforce | Nothing |

### 2. AWS Global Infrastructure
- **Region** — Physical location with multiple data centres (e.g., ap-south-1 = Mumbai)
- **Availability Zone (AZ)** — One or more data centres within a Region
- **Edge Locations** — Cache content closer to users (CloudFront CDN)

### 3. Core AWS Services (Exam Relevant)
| Service | What It Does |
|---------|-------------|
| EC2 | Virtual servers in the cloud |
| S3 | Object storage (files, data) |
| RDS | Managed relational databases |
| Lambda | Run code without servers |
| IAM | Manage access & permissions |
| VPC | Private network in AWS |
| CloudTrail | Track all API activity (audit logs) |
| CloudWatch | Monitor metrics, logs, alarms |

### 4. AWS Pricing Basics
- **Pay-as-you-go** — Only pay for what you use
- **Save when you commit** — Reserved Instances = cheaper
- **Free Tier** — Many services free for 12 months
- **AWS Pricing Calculator** — Estimate costs before you build
