### GenAI Key Terms
| Term | One-Line Definition |
|------|-------------------|
| LLM | Large Language Model — AI for text |
| Foundation Model | Pre-trained large model for many tasks |

### ML Key Terms
| Term | One-Line Definition |
|------|-------------------|
| Supervised Learning | Learn from labelled data |
| Unsupervised Learning | Find patterns in unlabelled data |
| Reinforcement Learning | Learn by trial and reward |
| Overfitting | Memorises training data, fails on new |
| Underfitting | Too simple, misses patterns |
| Precision | Of predicted positives, how many correct |
| Recall | Of actual positives, how many found |
| F1 Score | Balance of Precision + Recall |
| Bias | Unfair model skewed toward certain groups |
| Drift | Model accuracy drops over time in production |

## 📅 Exam Day Tips

### 1 Week Before Exam
- [ ] Complete 2 full practice tests (score 75%+ before booking)
- [ ] Review any topics where score is below 70%
- [ ] Re-read the cheat sheet above daily
- [ ] Book your exam slot at Pearson VUE or PSI

### Night Before Exam
- [ ] Light revision only — no new topics
- [ ] Sleep 7–8 hours
- [ ] Prepare ID proof (passport/Aadhaar)
- [ ] Check exam center location or online proctoring setup

### On Exam Day
- [ ] Arrive 30 min early (online: 15 min before)
- [ ] Read EVERY question fully — don't rush
- [ ] Eliminate wrong answers first (usually 2 can be removed)
- [ ] Flag difficult questions and come back
- [ ] Never leave a question blank — always guess if unsure
- [ ] Trust your first instinct — don't over-change answers

### Exam Strategy
```
85 questions × 120 minutes = ~1.4 minutes per question
First pass: Answer easy ones, flag hard ones
Second pass: Return to flagged questions
Final check: Review all before submitting
```
